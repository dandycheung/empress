onEvent

abstract fun onEvent(event: Event, model: Model<Patch>, requests: Requests<Event, Request>): Collection<Patch>

Handles an incoming event.

Parameters

event - An event that was triggered.

model - Current model.

requests - Allows to post new requests or cancel existing ones.

Return
A collection of updated patches. You should only return patches that have changed. If nothing has changed, you can return an empty collection. Based on updated patches, a new update will be sent.
