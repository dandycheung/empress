enthrone

fun <E : Empress<M, S>, M : Any, S : Any> FragmentActivity.enthrone(empressId: String, empress: E, eventDispatcher: CoroutineDispatcher = Dispatchers.Main, requestDispatcher: CoroutineDispatcher = Dispatchers.Default, retainInstance: Boolean = true): EmpressApi<E, M, S>

Installs an empress instance into the activity. If an empress with the same id was already installed, this method will return an existing instance.

Parameters

empressId - Id used to identify an EmpressApi instance.

empress - Instance of Empress to install.

eventDispatcher - A dispatcher to use for handling events in empress.

requestDispatcher - A dispatcher to use for handling requests in empress.

retainInstance - If true, the empress instance will be retained during configuration changes.

Return
An instance of EmpressApi for communicating with empress.
